# Grabby Spice
### A UserScript to download bots from SpicyChat

## Installation
As a prerequisite, a userscript manager is obviously required. I recommend [Violentmonkey](https://violentmonkey.github.io/).

### [Click Here to Install](https://github.com/Danbario-of-the-Void/Grabby-Spice/raw/refs/heads/main/grabspice.user.js)

## Known Issues
Currently, the script doesn't work unless you open a bot's info page in a new tab or refresh it.
